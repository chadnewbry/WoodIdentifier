# Wood Identifier

iOS app to identify wood species from photos using AI.

## Features

- **Camera View** — Take photos of wood for identification
- **Results View** — View identified wood species and properties
- **Wood Database** — Browse and search wood species
- **Project Tracking** — Track woodworking projects
- **Settings** — App preferences and information

## Tech Stack

- SwiftUI
- iOS 17+
- AI-powered identification via OpenAI proxy

## Project Structure

```
WoodIdentifier/
```
